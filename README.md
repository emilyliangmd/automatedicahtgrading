# automatedicahtgrading
Automated ICAHT grading by EHA/EBMT criteria using [heatwaveR](https://robwschlegel.github.io/heatwaveR/).

References:
1. Schlegel RW, Smit AJ (2018). “heatwaveR: A central algorithm for the detection of heatwaves and cold-spells.” Journal of Open Source Software, 3(27), 821. doi:10.21105/joss.00821.
2. Smit A (2023). Detect event streaks based on specified thresholds. https://tangledbank.netlify.app/blog/2023-11-22-run-lengths.
